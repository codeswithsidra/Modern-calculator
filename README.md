# Modern Calculator

A clean, responsive calculator built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Clean UI with hover effects
- AC, DEL, %, +, -, *, / support
- Error handling

## How to Use
1. Open `index.html` in any browser.
2. Start calculating!
